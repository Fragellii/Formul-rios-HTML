Formulário HTML - Padrões Web

Responda as perguntas abaixo

1- Na tag input com atributo type="text", para que serve o atributo "placeholder"?

O placeholder é um atributo do HTML que cumpre a função de fornecer ao usuário uma dica que descreve o valor esperado em um dado campo de entrada. Esta dica deverá ser apresentada ao usuário antes que ele entre com o valor e deverá ainda, ser curta e objetiva.

2- Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Os checkbox possuem valores boleanos, ou seja: “verdadeiro” ou “falso”. Dessa forma, ao atribuir o checked a um checkbox, o mesmo passa a ter um valor verdadeiro, inicialmente.

3- Na tag input para que serve o atributo "value"?

O atributo de valor pode ser usado para definir um valor padrão para um campo de entrada, definir valores para botões de opção, caixa de seleção, elementos de opção e definir texto de exibição de um botão

4- Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

input type="radio"

5- Para que serve a tag label? Para que serve o atributo "for" nessa tag?

Representa uma legenda para um item em uma interface de usuário. O atributo for serve para referenciar o id da tag input, deve conter o mesmo nome.

6- Na tag option para que serve o atributo "selected"?

É usado para criar um controle que representa um item dentro de um elemento HTML5 select, optgroup ou datalist.

7- Nas tags de formulário para que serve o atributo "required"?

É usado para indicar que a entrada do usuário é obrigatória em um elemento antes que um formulário possa ser enviado.

8- Na tag form para que serve o atributo "action"?

Define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.

9- Na tag form para que serve o atributo "method"?

Define qual o método HTTP para enviar os dados

10- Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

method="post". Nunca se deve usar o atributo GET se o seu formulário for conter dados confidenciais

11- Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

O type seria file.
